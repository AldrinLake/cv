# CV – Aldrin Lake

A clean, responsive single-page CV built with plain HTML and CSS.

## Live site

Enable **GitHub Pages** (`Settings → Pages → Deploy from branch: main, / (root)`) and the CV will be served at:

```
https://aldrinlake.github.io/cv/
```

## Files

| File | Purpose |
|---|---|
| `index.html` | CV content & structure |
| `style.css` | Styles (responsive, print-friendly) |

## Customising

Edit `index.html` to update personal details, experience, education, skills and projects.  
Edit `style.css` to change colours or layout.
